# Scholarly

**Scholarly** is an interactive web application designed to make learning collaborative, simple, and engaging. It helps students, educators, and professionals connect by asking questions, providing answers, and sharing notes.It is a small project built around a simple idea — making academic resources easier to manage, share, and actually use.
Most of the time, notes, links, PDFs, and references end up scattered everywhere. This project tries to bring everything into one place in a way that feels clean and usable.

---
## **What it does**

- Organizes study material in a structured way  
- Makes it easier to access and revisit resources  
- Focuses on a simple and distraction-free experience

## **Key Features**

* **Ask & Answer Questions:** Post questions and get answers from the community. Comment on answers to clarify or discuss further.
* **Upload & Access Notes:** Share your notes or download helpful resources uploaded by others.
* **Tag-Based Organization:** All content is categorized with tags so users can easily find relevant material.
* **Chat with PDFs:** Ask questions directly about uploaded PDF documents and get instant responses.
* **Clean & Responsive UI:** User-friendly interface optimized for ease of use.
* **Secure Environment:** Sensitive information like API keys is never committed to the repository.

  

##  Tech Stack

*Front-end : React.js
*Back-end : Node.js ,Express.js
*Database and storage : MongoDB,Cloudinary
*Microservice : Python
